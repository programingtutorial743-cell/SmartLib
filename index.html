📚 SmartLib

Python Library Management System

A lightweight desktop application developed by Nexa Code Lab to help schools manage books, borrowing records, and library operations.

⬇ Download Windows (.exe) ▶ Watch Demo

About

SmartLib is a Python-based desktop application that replaces manual library record keeping with an easy-to-use digital system. It allows librarians to organize books, manage borrowers, and keep accurate records efficiently.

Features

📖 Book Management

Add, edit and organize books easily.

🔍 Search

Quickly search books by title or author.

📥 Borrow & Return

Track borrowed and returned books.

💾 Offline Database

Uses SQLite for reliable offline storage.

🖥 Windows Desktop

Distributed as a standalone Windows executable.

Tkinter

Why SmartLib?
